# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in mcp-contracts, please report it responsibly.

**Do not open a public issue.** Instead, email **lukes.work@icloud.com** with:

- A description of the vulnerability
- Steps to reproduce
- Impact assessment (if known)

You should receive an acknowledgment within 48 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Scope

This policy covers:

- `@mcp-contracts/core` — the snapshot and diffing library
- `@mcp-contracts/cli` — the `mcpdiff` CLI tool
- The snapshot format (`.mcpc.json`)
